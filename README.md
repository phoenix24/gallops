sample play project build using sbt.
====================================


### usage
```
#### clean
$ /opt/sbt-0.13.5/bin/sbt clean

#### compile
$ /opt/sbt-0.13.5/bin/sbt compile

#### test
$ /opt/sbt-0.13.5/bin/sbt test

#### run play app
$ /opt/sbt-0.13.5/bin/sbt  'project gallops-dashboard' 'run'

#### run play app
```
